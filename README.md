# 📊 Estudos de SQL - Bootcamp GenAI&Dados 

Este repositório contém as práticas, consultas e conceitos desenvolvidos durante o módulo de Banco de Dados do Bootcamp GenAI&Dados em parceria com a DIO e o Bradesco.

Os conteúdos estão organizados de forma estruturada, acompanhando os tópicos abordados ao longo das aulas.

---

## 📂 Estrutura

O repositório está organizado por módulos e aulas, conforme os scripts desenvolvidos:

### 📌 Modelagem de Dados Relacionais
- 01 - Tabelas, Colunas, Registros e operações CRUD
- 02 - Alterando e Excluindo Tabelas
- 03 - Chaves Primárias e Estrangeiras
- 04 - Normalização de Dados

### 📌 Consultas Avançadas
- 05.1 - Consultas com Junções (JOINs) e Subconsultas
- 05.2 - Funções de Agregação e Agrupamento de Resultados
- 05.3 - Índices e Otimização de Consultas

---

## 🧠 Conteúdos abordados

- Modelagem de dados relacionais
- Criação e manipulação de tabelas (`CREATE`, `ALTER`, `DROP`)
- Operações CRUD (`INSERT`, `SELECT`, `UPDATE`, `DELETE`)
- Definição de chaves primárias e estrangeiras (PK e FK)
- Relacionamentos entre tabelas
- Normalização de dados
- Consultas com junções (`INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`)
- Subconsultas
- Funções de agregação (`COUNT`, `AVG`, `SUM`, `MIN`, `MAX`)
- Agrupamento de dados com `GROUP BY`
- Ordenação de resultados com `ORDER BY`
- Limitação de resultados (`LIMIT`, `OFFSET`)
- Criação de índices (`INDEX`)
- Análise e otimização de consultas com `EXPLAIN`

---

## 🚀 Objetivo

Consolidar conhecimentos em banco de dados relacionais e SQL, desenvolvendo habilidades essenciais para análise de dados e construção de sistemas.
